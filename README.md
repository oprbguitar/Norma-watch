## Funciones

| Función | Detalle | Plan |
|---|---|---|
| 🔎 **Buscador Legal Perú** | Busca normas, decretos y resoluciones en **El Peruano + gob.pe**, por últimos días o **rango de años** | Gratis (10 búsquedas/día) |
| 📡 **Monitoreo automático** | Vigila El Peruano cada hora con sus palabras clave (SST, contrataciones, presupuesto, MEF… editables) y notifica | Gratis |
| 📣 **Alertas correo/WhatsApp** | Deja el mensaje listo en Gmail / WhatsApp; usted solo confirma el envío | Gratis |
| 📄 **Exportación** | **PDF · Word · Excel · Markdown · JSON** en todas las vistas | Gratis |
| 🔍 **Interpretador de Normas** | Explica una norma en lenguaje claro: obligaciones, a quién aplica, plazos, sanciones y riesgos | 🔒 Premium |
| 📅 **Línea de Tiempo Legal** | Convierte una norma/expediente en línea de tiempo: publicación, vigencia, hitos, modificaciones | 🔒 Premium |
| ⚔️ **Comparador Legal IA** | Compara dos normas o versiones y explica los cambios artículo por artículo | 🔒 Premium |
| 📋 **Resumen diario + análisis de búsquedas** | Resumen ejecutivo diario y contraste inteligente de resultados; búsquedas ilimitadas | 🔒 Premium |

## Instalación rápida

1. **Descargue** el [ZIP](https://github.com/oprbguitar/Norma-watch/raw/main/docs/descargas/norma-watch-peru.zip) y descomprímalo.
2. Abra `chrome://extensions` → active **Modo de desarrollador**.
3. Pulse **Cargar extensión sin empaquetar** → seleccione la carpeta `extension`.
4. Configure sus palabras clave y canales en ⚙️ Opciones.

Guía paso a paso (con solución de problemas): **[INSTALACION.md](INSTALACION.md)**.
